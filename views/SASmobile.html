SASmobile

SASmobile is the unoifficial mobile application for the Semester at Sea study abroad program. Built with React Native, this is a cross-platform app that provides users with the ability to explore information about Semester at Sea's voyages, academic sponsors, ships, and ports of call.

Role and Responsibilities

1. Utilized the built-in Fetch API to connect the application with the Semester at Sea API

2. Integrated React-Native-Maps to interactively display information on each voyage's ports of call to a world map.

3. Used Redux's global store to control and keep track of the map component's state from view to view.

Technologies Used:

  • React Native
  • Redux
  • JavaScript
  • React Native Maps
