mood.fm

A web application that creates a playlist of music based on the weather in your city

mood.fm is a single-page web application built using Node.js and Express that captures location based on user input and outputs a custom playlist based on the weather in that location.

  • HMTL5
  • CSS3
  • JavaScript
  • Node.js
  • Express.js
  • EJS
  • OpenWeatherMap API
  • Spotify API
